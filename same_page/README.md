# Same Page RTC Transport Example

This example demonstrates how to use the `RtcTransport` API to send and receive data between two transports on the same page.

## How to run the example

1.  Open `same_page/index.html` in a browser.
2.  The two transports will be created and connected automatically.

## How to use the example

1.  Type a message in the first text area and click the "Send 1->2" button to send a message from the first transport to the second transport.
2.  Type a message in the second text area and click the "Send 2->1" button to send a message from the second transport to the first transport.
3.  The received messages will be displayed in the status area.
